# ERC20 协议理解笔记

## 1. ERC20 解决了什么问题？
ERC20 是一种代币标准，
它让钱包、交易所、协议都能
用统一的方式与代币交互。

## 2. 核心功能（非技术视角）
- transfer：代币转账
- approve / transferFrom：授权别人使用你的代币
- totalSupply / balanceOf：代币总量与余额查询

## 3. 权力结构分析
- 标准 ERC20 默认不包含增发功能
- 没有暂停、没有升级入口
- 合约部署后规则基本不可更改

## 4. 去中心化程度判断
- 代币规则是确定的、自动执行的
- 单一主体无法随意改余额
- 去中心化程度取决于是否添加额外模块

## 5. 风险认知
- 风险主要来自“非标准改动”
- 增发、暂停、可升级机制会引入中心化风险
